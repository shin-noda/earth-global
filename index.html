How to use <earth-globe>?

The <earth-globe> tag is a custom web component that displays an interactive 3D Earth. To use it, you must first include the following script inside your HTML <head>:

<script src="https://shin-noda.github.io/earth-globe/earth-globe.js" defer></script>
